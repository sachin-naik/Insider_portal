{{comp}} is a wrapper for native input and textarea elements. This container applies Material Design styles and behavior while still allowing direct access to the underlying native element.

The native element wrapped by the md-datepicker must be marked with the mdInput directive.

DatePicker

Html
<md-input-container>
<input mdInput [mdDatepicker]="picker" placeholder="Choose a date">
<md-input-container>
{{comp}} </md-datepicker>
