<md-menu> is a wrapper for native input and textarea elements. This container applies Material Design styles and behavior while still allowing direct access to the underlying native element.

The native element wrapped by the md-input-container must be marked with the mdInput directive.

Menu

Html
<button md-button [mdMenuTriggerFor]="menu">Menu</button
<md-menu #menu="mdMenu">
<button md-menu-item>Item 1 </button> <button md-menu-item>Item 2 </button> </md-menu>
